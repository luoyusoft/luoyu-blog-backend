## 主要功能：
1、文件上传、下载
2、Excel生成、解析
3、定时任务
4、消息推送
5、在线聊天
6、文章模块部分功能
7、用户模块
8、权限模块

## 技术栈：
前端：JavaScript、Vue.js、Vuex、Vue Router、Element-UI、Axios、Stomp.js、Node.js。
后端：SpringBoot、Mybatis Plus、Apache Shiro、JSON Web Token、WebSocket、Stomp、Quartz、RabbitMQ、RocketMQ、Swagger、Maven。
数据库：MySQL、Redis。
服务器：Centos7、Nginx。

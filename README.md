## 项目简介
```
基于 SpringBoot + Vue 搭建的前后端分离个人博客后台管理系统的后端部分
```

## 主要功能
```
博文，图书，笔记的新增，删除，修改，查看，推荐，置顶，发布
分类管理，标签管理，友链管理，推荐管理
管理员管理，角色管理，菜单管理，系统参数，SQL监控
```

## 前端技术栈
```
Vue 2.x
Vuex
Vue Router
Vue-cli
Element-UI
```

## 后端技术栈
```
SpringBoot 2.x
Mybatis-Plus
Apache Shiro
Xxl-Job
Elasticsearch
Redis
MySQL
RabbitMQ
七牛云
knife4j
Lombox
```

## 前端部署
```
# 安装依赖
npm install

# 启动项目
npm run dev

# 构建开发环境
npm run build:dev

# 构建生产环境
npm run build:prod
```
